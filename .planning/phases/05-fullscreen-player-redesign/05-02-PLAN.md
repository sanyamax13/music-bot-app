**Change 3 — Add double-tap detection at end of hydrate():**

After the volume slider event listeners block (after the `if (volSlider) { ... }` block), and before the closing `}` of hydrate(), add:

```javascript
// Double-tap like on cover (FP-7, D-03/D-04/D-05)
let doubleTapTimer = null;
const coverWrap = root.querySelector('.fp-cover-wrap');
if (coverWrap) {
  coverWrap.addEventListener('click', (e) => {
    // Do not intercept scrubber drags or button taps
    if (e.target.closest('input, [data-action]')) return;
    if (doubleTapTimer) {
      clearTimeout(doubleTapTimer);
      doubleTapTimer = null;
      // Show heart overlay animation
      const overlay = root.querySelector('.fp-heart-overlay');
      if (overlay) {
        overlay.classList.remove('animate');
        void overlay.offsetWidth; // force reflow to restart animation
        overlay.classList.add('animate');
        setTimeout(() => overlay.classList.remove('animate'), 600);
      }
      // Haptic feedback (D-05)
      if (window.Telegram?.WebApp?.HapticFeedback) {
        Telegram.WebApp.HapticFeedback.impactOccurred('light');
      }
      // Optimistic like — same pattern as case 'like' (lines 1304-1318)
      const t = Store.get('currentTrack');
      if (!t) return;
      const heart = root.querySelector('#fp-fav-btn');
      const liked = Store.get('likedIds');
      if (heart) { heart.innerText = '\u2764\uFE0F'; heart.style.transform = 'scale(1.3)'; setTimeout(() => heart.style.transform = '', 200); }
      liked.add(t.id);
      Store.set({ likedIds: new Set(liked) });
      Api.like(t).catch(() => {
        liked.delete(t.id);
        Store.set({ likedIds: new Set(liked) });
        if (heart) heart.innerText = '\uD83E\uDD0D';
      });
    } else {
      doubleTapTimer = setTimeout(() => { doubleTapTimer = null; }, 300);
    }
  });
}
```

IMPORTANT per D-04: The 300ms setTimeout distinguishes single-tap (no action on cover) from double-tap (like). Single tap on cover area does nothing, which is correct — the cover itself has no single-tap action.

IMPORTANT: The `e.target.closest('input, [data-action]')` guard prevents double-tap from firing when the user taps buttons or the scrubber inside the cover area. The `[data-action]` guard also prevents the `search-artist` tap on `#fp-artist` from being captured as the first tap of a double-tap sequence.
