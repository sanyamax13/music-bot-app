# Phase 5 — UI Design Contract

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage in this phase |
|-------|-------|---------------------|
| xs | 4px | Time label margin-top (fp-time), icon gaps |
| sm | 8px | Handle margin-bottom, queue-peek vertical padding |
| md | 16px | Meta section margin-top, controls margin-top |
| lg | 24px | Horizontal page padding (left/right), border-radius on cover |

| 2xl | 40px | Feedback button gap (existing: 40px, keep) |

Exceptions: Touch-target minimum 44px on scrubber thumb tap area (accessibility requirement, not a spacing token).

## Interaction Contracts

### Double-Tap Like (FP-7, D-03/D-04/D-05)

| Property | Value |
|----------|-------|
| Gesture | Two taps within 300ms on `#fp-cover` |
| Detection | `setTimeout` 300ms after first tap; second tap within window cancels timer and fires like |
| Single-tap on cover | No action (timer expires, nothing happens) |
| Visual feedback | `.fp-heart-overlay` appears centered on cover: scale(0) -> scale(1.2) -> scale(1) -> opacity 0, total 600ms |
| Haptic | `tg.HapticFeedback.impactOccurred('light')` if available |
| State | Triggers same optimistic like flow as `#fp-fav-btn` tap |
| If already liked | Animation still plays (no unlike on double-tap, matches Instagram pattern) |

#### Heart Overlay CSS

```css
.fp-heart-overlay {
    position: absolute;
    top: 50%; left: 50%;
    transform: translate(-50%, -50%) scale(0);
    font-size: 72px;
    pointer-events: none;
    z-index: 10;
    opacity: 0;
}
.fp-heart-overlay.animate {
    animation: fp-heart-pop 600ms cubic-bezier(0.2, 0.8, 0.2, 1) forwards;
}
@keyframes fp-heart-pop {
    0%   { transform: translate(-50%, -50%) scale(0); opacity: 1; }
    30%  { transform: translate(-50%, -50%) scale(1.2); opacity: 1; }
    50%  { transform: translate(-50%, -50%) scale(1); opacity: 1; }
    100% { transform: translate(-50%, -50%) scale(1); opacity: 0; }
}
```

Content: emoji `\u2764\uFE0F` (red heart).

### Queue-Peek (FP-8, D-06/D-07/D-08)

| Property | Value |
|----------|-------|
| Visibility | Shown when `queue[queueIndex + 1]` exists; hidden otherwise |
| Store subscription | `['queue', 'queueIndex']` |
| Content | `Далее: {title}` |
| Overflow | `text-overflow: ellipsis; white-space: nowrap; overflow: hidden` |
| Max width | 80% of container, centered with `margin: 0 auto` |

#### Queue-Peek CSS

```css
.fp-queue-peek {
    font-size: 12px;
    color: var(--text-muted);
    text-align: center;
    margin-top: 8px;
    max-width: 80%;
    margin-left: auto;
    margin-right: auto;
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
}
```

### Tap Artist -> Search (FP-9, D-09/D-10)

| Property | Value |
|----------|-------|
| Trigger | Tap on `#fp-artist` |
| Behavior | 1) Close fullscreen player 2) Navigate to Search tab 3) Prefill search input with artist name 4) Auto-trigger search |
| Visual hint | `#fp-artist` gets `cursor: pointer; text-decoration: underline` on hover/focus (desktop); no underline on touch (rely on tap feedback) |
| Tap feedback | `:active` opacity 0.6 on `#fp-artist` |

### Cover Resize (FP-1, D-01/D-02)

```css
#fp-cover {
    width: 70vw;
    max-width: 340px;
    aspect-ratio: 1/1;
    border-radius: 24px;
    /* rest unchanged */
}
```

### Scrubber Touch Target (FP-3, D-11)

Expand touch area without changing visual appearance:

```css
input[type=range]::-webkit-slider-thumb {
    width: 14px; height: 14px;
    /* Add transparent border to expand hit area to 44px */
    border: 15px solid transparent;
    background-clip: content-box;
    box-sizing: content-box;
}
```

Alternative: wrap `input[type=range]` in a div with `padding: 12px 0` to reach 44px total height.

## Cover Container (for double-tap overlay)

The `#fp-cover` img must be wrapped in a positioned container to anchor the heart overlay:

```html
<div class="fp-cover-wrap" style="position: relative; display: flex; justify-content: center;">
    <img id="fp-cover" ...>
    <div class="fp-heart-overlay"></div>
</div>
```

This is the only template structure change. The wrapper inherits margin/centering from the current `#fp-cover` auto margin.

## Copywriting Contract

| Element | Copy (Russian) |
|---------|----------------|
| Queue-peek prefix | `Далее:` |
| Queue-peek when no next track | Element hidden (no text, `display: none`) |
| Artist tap hint | None (no tooltip; behavior is discoverable) |

No empty states, error states, or destructive actions in this phase. The fullscreen player is always opened with a valid `currentTrack`.

## Animation Budget

| Animation | Duration | Easing | Property |
|-----------|----------|--------|----------|
| FP open/close | 400ms | `cubic-bezier(0.33, 1, 0.68, 1)` | `top` (existing) |
| Dominant-color bg transition | 500ms | `ease` | `background` (existing) |
| Heart overlay pop | 600ms | `cubic-bezier(0.2, 0.8, 0.2, 1)` | `transform, opacity` (new) |
| Like button scale | 200ms | default | `transform` (existing) |
| Feedback button scale | 200ms | default | `transform` (existing) |

All animations use `transform` and `opacity` only (GPU-composited, no layout thrash). Target: 60fps on budget Android.
